1 = Step to create a project in django 
          first make vertual environmet to create any project in django

2 = On that environmet install all neccessary tools by using pip like django djangoFramework etc.

3 = now create project in django using command = django-admin startproject <project-name>

4 = after creating project then create app = python manage.py startapp <app_name>

5 = after creating app then add this app in project/setting.py file where written INSTALL APP[...,app-name,]

point to be noted that in project folder there is file that is urls.py but in app folder there is no such this file\
so we have to create first urls.py in app folder just copy from project/urls.py and past app/urls.py

now from starting  the user enter url on the browser that url match with project urls.py

step to creating models= >

step 1 -> go in your app and find the file name models.py Now here you have create
schema of your database like tables and constrains

syntax =>

from django.db import models

class Model_Name(models.Model):
    # Field definitions go here
    field_name = models.FieldType(options)

example+>

class Product(models.Model):
    product_id = models.AutoField(primary_key=True)
    product_name = models.CharField(max_length=50)
    category = models.CharField(max_length=50,default="")
    subcategory = models.CharField(max_length=50,default="")
    price = models.IntegerField(default=0)
    desc = models.CharField(max_length=300)
    pub_date=models.DateField()
    image = models.ImageField(upload_to="shop/images", default="")
    
    def __str__(self):// for show name of the prodect in databse when something is added
        return self.product_name
